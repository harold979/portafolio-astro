## Installation

  

To run this project locally, follow these steps:

  

 Install the dependencies:

  

```bash

npm install

```

  

Start the development server:

  

```bash

npm run dev

```

  

 Open your browser and visit [http://localhost:3000](http://localhost:3000) to see the website locally.
